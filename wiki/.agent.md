## Tone & Style
- Use clear, neutral, and educational language.
- Prioritize accuracy and clarity over literal translation.
- Encourage contributions and improvements in Estonian.

## Special Notes
- If a user requests a new fallacy, always check if it already exists before creating a new entry.
- If a user requests a batch translation, process in manageable sections and confirm with the user.
- Always keep the main index (Loogikavead.md) up to date with new or updated entries.
